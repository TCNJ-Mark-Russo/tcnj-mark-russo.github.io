From a user’s perspective, DoodlePad is nothing more than a zip archive file containing precompiled Java classes ready for use packaged as a Java Archive (JAR) file. No installation is required other than to download the doodlepad.jar file from the project’s GitHub repository 1 . You will find a direct link to the GitHub repository and the JAR file at the site https://doodlepad.org/ 2 .
