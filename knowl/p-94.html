Any time we enter a number directly into a Java program, even if we don’t store it using a variable, Java must interpret it and hold in memory, at least temporarily. Whenever any value is entered directly into a Java program, such as a number, it is called a literal. Integer literals of type int are written using no special syntax, but other types require extra trailing characters to indicate their type.
