Unlike base classes which are extended by another class, interfaces are implemented by a class. Otherwise, an interface can be thought of an abstract base class with nothing but abstract method signatures. Implementing a interface can be thought of as a contract that must be fulfilled by the implementing derived class. To fulfill the contract, a class that agrees to implements an interface must implement all the signatures in the interface. Like with abstract methods, the Java compiler enforces the contract, raising a compile-time errors if an interfaceâ€™s signature is not implemented.
