The ToggleButton does not implement setFontSize(…) and setFontStyle(…), but it does inherit these methods from its base class. Modifying main(…) and rerunning our program, we see the update output in Figure 10.3.4. This would have taken some work ahd we stuck with our old implementation. We would have had to implement all new methods that modified the encapsulated RoundRect. With inheritance, we get the behavior automatically. No modifications required.
