Inheritance: Inheritance allows the creation of new classes based on existing classes, inheriting their attributes and behaviors. The existing class is called the "parent class", "base class", or "superclass", and the new class is called the "child class","derived class", or "subclass". The child class can extend or modify the behavior of the parent class by adding new methods or overriding existing ones. Inheritance promotes code reuse and supports the concept of hierarchical relationships between classes.
