When testing Strings having an even number of chars, for example when lets.length() == 6, then the index of the first char is 0 and the last is 5. In other words, when i == 0, j = 6-1-0 = 5; we are comparing the first and last chars of the String. When i == 1, j = 5-1-1=4, and when i == 2, j == 3. The pair of indexes of chars to compare are calculated correctly.
