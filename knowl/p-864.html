In this section we want to build a graphical panel demonstration three kinds of tri-option selector widgets. The way these widgets look may be customized, but all must give the user the ability to select from the three options and provide a way to report the current option selected. To ensure that all tri-option widgets provide common capabilities, we define a TriOption interface that all selector widgets must implement.
