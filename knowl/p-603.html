Without getting into to much detail, you know that data is store in memory in binary, in other words base-2 numbers (1s and 0s). But humans work (mostly) in decimal, base-10 numbers. It is not possible to store many base-10 numbers exactly using base-2 notation, especially when we have limited memory. For this reason, floating point number operations produce base-2 results that are as close as possible to the real base-10 numbers, given the precision possible in the available memory.
