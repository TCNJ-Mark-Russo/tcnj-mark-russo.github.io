As we know, inheritance provides derived classes with access to all base class members. So, what do we get from Object? After all, if all classes inherit Object, then any non-private members that we find will be available to all of our classes. A little sleuthing uncovers the OpenJDK source code for the Object class 1 . The source code for two inherited Object class methods are of particular interest. These are toString() and equals(…).
