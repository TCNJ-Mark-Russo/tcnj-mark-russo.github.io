Two additional unary operators require a little more explanation. These are the increment operator (++) and the decrement operator (--). Both of these operators may be used in unary prefix and unary postfix form on a variable, but not on a primitive number. The ++ increment operator simultaneously adds a value of 1 to the existing value of a variable and reassigns the variable to the incremented value. The -- decrement operator subtracts a value of 1 from the existing variable value and reassigns it.
