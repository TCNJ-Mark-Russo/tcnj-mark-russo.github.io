There are four keywords to identify: switch, case, break, and default. The switch keyword is followed by parentheses containing a selector expression, and a code block delimited by brackets. The selector expression must evaluate to a number, String, or other constant. Each label following a case identifies the number or String to be matched against the value obtained from the evaluated selector expression. If a match is found with a case label, the statements following the case are executed. If/when a break is encountered, the switch statement is exited. If the default branch is encountered, it executes automatically. No match is required.
