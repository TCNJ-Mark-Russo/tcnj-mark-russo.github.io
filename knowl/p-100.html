It’s important to note that variables have a scope, which determines where they can be accessed within the program. Variables can have block scope (limited to a specific block of code), method scope (limited to a specific method), object (instance) scope (limited to a specific object), class scope (accessible throughout the class), and package scope (limited to a Java package). The scope of a variable is defined by the location where it is declared. We’ll explore this topic in more detail later. For now, understand that the location in your program where a variable is declared dictates from where it can be accesses.
