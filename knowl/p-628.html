Once created, you may access and use the elements of an array just like you would a single-valued variable. The only difference is that you must follow your array variable with square brackets and an item index so that Java knows which element of the array that you want to access. In the example below, the data array variable is declared and initialized to a new array of doubles with length 100. These items are then accessed by following the array variable name data with square brackets and an index value, like data[0].
