Item 14.

To cast data from one type to another, precede the given value, variable, or expression with parentheses containing the new type. For example, to convert a long to an int, precede a variable, value or expression with (int). 
jshell> (int)123L
$2 ==> 123
| created scratch variable $2 : int
To convert a double to a float, precede a variable,
value, or expression with (float)
jshell> (float)12.34
$3 ==> 12.34
| created scratch variable $3 : float
