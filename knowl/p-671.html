The program in Listing 8.3.1 defines three static methods. In addition to the main(…) method you will find the printArr( double[] arr ) method and the doubleArr( double[] arr ). Both methods take a double array parameter. The printArr(…) method iterates over all elements of the array and prints each. The doubleArr(…) method multiplies all elements by 2 and reassigns each updated value back to the array element.
