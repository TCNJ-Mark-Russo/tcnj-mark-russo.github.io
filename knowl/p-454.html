The Counter class implements enter(), exit(), and reset() methods, which are used to manage the value of the total variable. When we access the total variable we precede it the with the name of the class, for example Counter.total. This ensures that we are accessing the class scope and the variables defined within it. We also invoke the class methods with a similar explicit setting of class scope, namely, Counter.enter() and Counter.exit(). While you will find that this explicit scoping is not necessary, it helps to communicate that we intend to access the class scope.
