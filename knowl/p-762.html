In which class should the overall main(…) method be placed — the one that initializes all classes and starts running your program? It may not always be obvious. After all, for our individual class definitions to be reusable, it does not make a lot of sense to use a class’s main(…) method to start a larger application. Furthermore, in Section 9.3 we used the main(…) method to run tests on our custom class.
