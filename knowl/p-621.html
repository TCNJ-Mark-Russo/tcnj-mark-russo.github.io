We know that for a logical conjunction to be true, both the left and right subexpressions must evaluate to true. Proceeding left-to-right, if we find that the left subexpression of a conjunction evaluates to false there is no need to evaluate the right subexpression because we know already that the entire conjunction is false. Java skips the right subexpression and return false as soon as it finds the that left subexpression evaluates to false. This is called short-circuit evaluation.
