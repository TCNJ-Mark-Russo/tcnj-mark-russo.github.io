Listing 8.4.2 demonstrates these ideas. This program has two method defined, main(…) and print2D(…). Both methods use the nested iteration approach to access array elements. In main(…) we access each element and initialize values to random doubles. In print2D(…) we access each element for printing. Pay special attention to the inner loop continuation test, j<arr2D[i].length. We are querying each inner array for its length, each array at arr2D[i]. In the next section we’ll learn why we want to ask each inner array for its length, and not just the first, or use a constant.
