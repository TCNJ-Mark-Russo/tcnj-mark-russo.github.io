This book was born from materials developed to teach many sections of the course CSC 220: Computational Problem Solving at The College of New Jersey 2 . Rather than start with the Java compiler, this book introduces students to the interactive Java Shell named JShell which allows them to explore primitive types, basic objects, and expression, all in an interactive context. We avoid the magical public static void main(String[] args) {... in favor of core topics such as types and memory.
