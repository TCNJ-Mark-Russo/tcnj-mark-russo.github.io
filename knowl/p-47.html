One of the real benefits of object oriented programming is abstraction. When I drive a car, I don’t need to know all the underlying technical details about how the car is able to move. I know that if I press the start button in my electric vehicle, then engine turns on. I know if I put the car in gear and press the accelerator pedal, the car moves. There is quite a lot of engineering that makes this possible, but I don’t need to know all the details.
