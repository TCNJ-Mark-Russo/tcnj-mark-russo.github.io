Branching is the programming term used for deciding to execute different parts of a program based on some condition. If running a program is like walking down a road, branching is like coming to a fork in that road and deciding which way to go. if-statements are the most common way to branch execution of a program, and both relational expressions 2.10 and logical expressions 2.11 are often how we make the decision about which way to go.
