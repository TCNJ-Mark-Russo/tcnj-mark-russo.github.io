When applied to a variable as a prefix operator, the result is what you would expect. Consider the following JShell session. The variable a is declared and initialized to a value of 5. After the prefix ++ is executed, the value changes to 6. Following this with the -- operator, the value is decrements and is reset back to 5. Nothing unusual there. Repeat this example yourself by declaring and initializing new variables, and incrementing and/or decrementing them using the prefix ++ and -- operators.
