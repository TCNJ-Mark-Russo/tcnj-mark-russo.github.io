  1. Debugging is the process of identifying and correcting errors in a program
  2. The simplest form of debugging a program involved hand tracing its statements and program flow.
  3. Another form of debugging involves strategically placed print statements in a program that reveal its intermediate state during execution.
  4. Java assertions are statement that evaluate an expression that should evaluate to true. An AssertionError is thrown when the expression evaluates to false.
  5. A debugger is special software what allows you to step through a program and introspect its state and values between each step.
