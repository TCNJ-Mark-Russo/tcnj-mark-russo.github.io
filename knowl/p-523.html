In the following code snippet, we declare and fill and ArrayList<Integer> with 100 randomly generated integers. Once filled, we loop over the ArrayList<Integer> again, get each int at each successive index and print it. In both cases (adding and getting) we do not box or unbox the primitive type int. Java automatically boxes each int in an Integer before adding to the ArrayList, and unboxes the Integer to an int when getting and assigning to the primtiive variable.
