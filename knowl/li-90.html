Item 9.

Java will operate only on operands having the same type. For example, Java will not add an int and long. Similarly, Java will not add a float and a double. Before evaluating an expression with different operand types, Java attempts to promote the type automatically of one of the operands to match the type of the other. These type changes are called widening conversions because more memory is used to store the converted data compared to the original data (the memory required widens).
