Two of these operators require a little more explanation: the increment operator (++) and the decrement operator (--). Both of these operators may be used in prefix and postfix form on a variable. The ++ increment operator simultaneously adds a value of 1 to the existing value of a variable and reassigns the variable to the incremented value. The -- decrement operator subtracts a value of 1 from the existing variable value and reassigns it. Why are there both prefix and postfix versions?
