The following program demonstrates how to instantiate a Scanner object and invoke its methods. We add an initial interactive step to our first Java Program in Listing 3.2.2. Once again, we are using the distanceBetween() method twice to compute and sum the distance between the first two points and the second two points. Don’t forget to close() the Scanner to avoid a resource leak. Closing a Scanner sets a flag in the object so other parts of your code knows it is not safe to read, it closes any underlying resource that it may be using, and it also makes it suitable for garbage collection to avoid memory leaks, a concept that we will investigate in more detail later. Also note that you may close an already closed Scanner multiple times -- no harm is done.
