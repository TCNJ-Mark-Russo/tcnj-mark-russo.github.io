When either of these operators are applied to a variable in prefix form, the result is what you would expect. The expression evaluates to the incremented or decremented values. Consider the following JShell session. The variable a is declared and initialized to a value of 5. After the prefix ++ is executed on a, the value changes to 6. Following this the prefix -- operator is applied, and the value is decrements a back to 5. Nothing unusual here. Repeat this example yourself by declaring and initializing new variables, and incrementing and/or decrementing them using the prefix ++ and -- operators.
