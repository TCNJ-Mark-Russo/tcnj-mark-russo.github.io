Relational expressions include the familiar greater-than (>), less-than (<) and equal-to (==) operators. Recall that the single-equal operator (=) is reserved for assignment, in Java. To test for equality, Java uses the double-equals operator (==). In addition to these operators, Java implements compound relational operators, including greater-than or equal-to (>=), less-than or equal-to (<=) and not-equal-to (!=). These have the obvious interpretations.
