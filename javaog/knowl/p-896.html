When testing Strings having an even number of chars, pairs of character indexes to compare move inward toward each other and meet at adjacent points. For example when lets.length() == 6, the index of the first char is 0 and the last is 5. In other words, when i == 0, j = 6-1-0 = 5; we are comparing the first and last chars of the String. When i == 1, j = 5-1-1=4, and when i == 2, j == 3, and our iteration is complete.
