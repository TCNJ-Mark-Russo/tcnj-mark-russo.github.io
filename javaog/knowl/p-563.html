Managing a single data value using a variable is fundamental to programming. But when we have too many items to track, the effort required can become too tedious, error-prone, or impractical. When our data values are related and of the same data type, we have a better option. We can store all data items in an array data structure and assigned it to a single variable name, but address each item in the array individually, using an associated index. In this chapter we will learn how to create, initialize, and work with arrays.
