Listing 12.4.1 draws a Fractal Tree, with the recursive method branch(…). The branch(…) method takes six parameters:
  1. x1: Branch starting x-location
  2. y1: Branch starting y-location
  3. angle: Angle with which to rotate branch [degrees]
  4. length: Length of branch
  5. dangle: Amount to rotate next branch (positive and negative) [degrees].
  6. dlength: Amount to shorten next branch [pixels]
The branch(…) method calls itself twice, with new angles and a shorter length. This method is an example of multiple recursion due to the two recursive invocations. The base case stops recursion when branch length is reduced to a value that is less than 10. Figure 12.4.2 shows output from Listing 12.4.1.
