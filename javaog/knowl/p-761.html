One of the limitiations to note is that the Java compiler checks that every method invocation on a class is implemented by the declared class type. This means that while we can assign a ToggleButton to a Shape variable, we cannot invoke its toggle() method. The compiler will stop with an error telling us that the Shape does not implement toggle(). The compiler does not run the program and so it cannot know that our shp variable actually references a ToggleButton object, so when it is run, the program will not have an error.
