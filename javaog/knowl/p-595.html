We follow the rules outlined above to create and initialize a new array that itself holds arrays. If double[] is the type of a double array formed by adding a pair of square brackets to the right of the type double, then an array of double arrays is obtained by adding another pair of square brackets after the double array, like so double[][]. Continuing with line of reasoning, if the expression new double[5] creates a new double array with 5 items, then the expression new double[5][3] creates a new array capable of holding 5 double arrays, each of which can hold 3 doubles.
