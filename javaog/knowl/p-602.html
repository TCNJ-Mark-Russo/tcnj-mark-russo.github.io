After declaration of the String array variable and creation of the String array, JShell prints the array for us. We see that the three elements of the String array are null, null, null. Recall that the null keyword signals the absence of an object. The names array contains no objects, whatsoever. True, it has the ability to reference three String objects should we create and assign them, but unless the array elements are initialized, we have completed only two of the three steps required for fully creating and populating an array: the array is created, but its elements have not been initialized.
