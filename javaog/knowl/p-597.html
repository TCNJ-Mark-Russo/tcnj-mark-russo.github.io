After declaration of the String array variable and creation of the String array, JShell prints the array for us. We see that the three element of the String array are null, null, null. Recall that the null keywords signals the absence of an object. The names array contains no objects, whatsoever. True, it has the space to reference three String objects should we create and assign them, but unless the array elements are initialized, we have completed only two of the three steps required for fully building out an array: the array is created, but its elements have not been initialized.
