Note how the super keyword can invoke an alternative constructor in a base class in a manner directly analogous to the way we used the this keyword to invoke an overloaded alternative constructor in the current class. Similarly, we are able to use the super keyword to access the scope of the base class directly, in a manner directly analogous to the way we used the this keyword to access the current object scope. The this and super keywords should occupy adjacent places in your Java memory bank.
