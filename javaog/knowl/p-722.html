First and foremost, inheritance solves the code duplication problem. If a method in a base class needs updating, any changes made are inherited automatically by all derived classes, no extra effort required. This reduces the chance for errors caused by neglecting to make necessary changes to all implementations of a shared method. Of course, if an error is made in the base class method, all derived classes inherit the same broken method. But fixing that error once also means that the corrected method is fixed everywhere.
