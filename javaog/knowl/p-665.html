Colors in DoodlePad may be partially or fully transparent. This is set using fourth value in [0, 255] known as alpha. An alpha of 0 means fully transparent and an alpha of 255 means fully opaque. All color-related DoodlePad methods can take four color component parameters, the fourth being transparency specified as alpha. If that fourth parameter is omitted, DoodlePad substitutes an alpha of 255, fully opaque. This is implemented by overloading the definition of setFillColor(…), which you can observe in the DoodlePad source code 1 .
