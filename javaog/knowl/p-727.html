Inheritance is a relationship that we can established between classes. When one class inherits another, the inherited class can be thought of as providing a kind of default implementation for the class that inherits it. Inherited classes are often referred to a base classes or superclasses and inheriting classes are called derived classes or subclasses. We’ll use the former term. In Java’s implementation of inheritance, a base class may be related to one or more derived class, but a derived class may inherit at most one base class.
