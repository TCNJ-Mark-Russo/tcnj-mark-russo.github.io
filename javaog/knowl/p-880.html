Find a complete example of reading a plain text file line-by-line in Listing 11.4.2. A BufferedReader returns null from its readLine() method when there are no more lines to read. With this in mind, pay close attention to the way the while-statement in Listing 11.4.2 is constructed. We invoke readLine() once before the while-statement, and enter the statement only if the line value is not null. This prevents further action to be taken on an empty file. Also, we read the next line at the end of the while-statement body to check if there is another line to be processed, and continue the while-statement only if there is another line.
