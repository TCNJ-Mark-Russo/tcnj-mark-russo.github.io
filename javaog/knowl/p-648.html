Let’s add a main(…) method to test our initial Java LED class. It is convenient to add a main(…) method to a class for testing. As we’ll see later, as our programs grow we rarely invoke an individual class’s main(…) method in favor of using a special driver class with a main(…) method designed specifically to start up a program. To test our class, in a main(…) method we create two LED instances at different locations. See Listing 8.2.4 and Figure 8.2.5. As expected, in Figure 8.2.5 two LED objects are shown at the given starting locations with a dark gray fill color.
