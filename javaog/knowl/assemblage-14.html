Rule: Sequence Data Structure Indexes Start at 0.

In Java, we learned that when accessing an indexed sequence of items, the index of the first item always starts at 0. This is the case for arrays as well. Think of an index as an offset from the start of a sequence, not as an itemâ€™s sequence position. The offset from the first element to the first element is 0, hence the index of the first element is 0.
