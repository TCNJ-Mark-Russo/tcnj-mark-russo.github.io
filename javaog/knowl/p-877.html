An alternative way to read plain text files in Java is to use the FileReader and BufferedReader classes. The FileReader class reads files byte-by-byte. This is not useful if we are interested in reading a file line-by-line. The BufferedReader class will wrap a FileReader object and use it to buffer file contents and the return these contents in a more convenient manner.
