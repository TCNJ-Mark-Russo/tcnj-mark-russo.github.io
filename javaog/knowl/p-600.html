Note how we use nested curly bracket to initialize the array, and how we don’t specify the dimensions of the nested arrays. Java determines the array dimensions by the structure of the initialization. Also note the type of the variable in the outer for-each loop inside printArr(…), which is for (double[] inarr : arr) {. The loop variable is a one-dimensional double array! By now, this should make sense. In Java, a multidimensional array is an array of arrays; the outer array is a one-dimensional array with elements that each hold one-dimensional arrays.
